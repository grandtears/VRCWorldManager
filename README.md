# VRC World Manager (VRChat World Manager)

**VRC World Manager** は、VRChat のワールドを直感的に管理・検索できる Windows 用デスクトップアプリです。ポータブル版として単一の `.exe` で動作し、インストール不要で持ち運びが可能です。

## ✨ 主な機能

- **💾 手動保存ボタン**: ヘッダーに常に表示され、タグ付けやお気に入りの変更を即座にサーバーへ同期できます。
- **🔎 ワールド ID 検索**: 検索バーに `wrld_...` 形式の ID を入力するだけで、特定のワールドを直接取得できます。
- **⭐ お気に入りの更新順ソート**: お気に入りワールドは VRChat 側の `updated_at` 降順で自動ソートされ、最新の更新が常に上位に表示されます。
- **🌙 ダークモード**: UI 全体でテーマ切替が可能。設定は永続化され、次回起動時に復元されます。
- **📁 JSON データのインポート/エクスポート**: 設定・カスタムリスト・タグ情報を JSON でバックアップ・復元できます。
- **🔐 セキュリティ強化 (DPAPI)**: Windows Data Protection API を利用したセッション情報の暗号化フォールバックを実装。Electron 以外で API サーバ単体起動時でも安全に保存されます。
- **🛠️ カスタムリスト・タグ管理**: 任意のリストやタグを作成し、ワールドを自由に分類できます。

## 🚀 使い方 (Getting Started)

1. **ダウンロード**: `VRCWorldManager.exe` を任意のフォルダに配置します。
2. **起動**: exe をダブルクリックして起動します。
3. **ログイン**: VRChat アカウントでログインし、2FA（メール・TOTP）に対応します。
4. **操作開始**: ワールド一覧が表示され、検索・タグ付け・お気に入り・カスタムリストの管理が可能です。

### データ保存場所
- 設定・カスタムリストは `settings.json` に保存されます。
- ログインセッションは `sessions.json` に暗号化して保存されます（DPAPI 使用）。
- ポータブル版なので、同フォルダにあるファイルをコピーすれば別 PC へ簡単に移行できます。

## 🛡️ セキュリティ
- **通信**: VRChat 公式 API のみへリクエストを送信し、外部サーバーへはデータを送信しません。
- **暗号化**: Windows DPAPI によりセッション情報をローカルで暗号化保存。Electron の `safeStorage` が利用可能な環境では自動的に暗号化されます。
- **オープンソース**: ソースコードは MIT ライセンスの下で公開され、透明性を確保しています。
- **注意**: ロケーションと招待のログから同一インスタンスのユーザーを取得していますが、VRChat のログはリアルタイムではなく、最新のログから取得できても実際のメンバーと差異が生じる可能性があります。

## 🛠️ 技術仕様 (Technical Specifications)

- **フロントエンド**: `apps/web` – React 18, Vite, TypeScript
- **バックエンド**: `apps/api` – Hono (Node.js), TypeScript
- **デスクトップ**: `apps/electron` – Electron, electron‑builder

### ビルド方法
```bash
# 依存関係のインストール
npm install

# ビルド (Web, API, Electron)
npm run build
```
ビルド成果物は `apps/electron/release/` に出力されます。
